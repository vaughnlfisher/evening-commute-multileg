# Evening Commute Multileg

Return-journey Home Assistant integration + Lovelace card.

**Route:** City Thameslink → Farringdon (Thameslink) → Paddington (Elizabeth line) → Twyford (GWR / Elizabeth line)

3-level anchored nesting: each leg shows only connections catchable after the previous leg arrives, plus interchange time (5 min at Farringdon, 8 min at Paddington). Shows the next 3 City Thameslink departures from 16:00, each with up to 3 onward Elizabeth line options, each with up to 3 onward Twyford services.

Data: Huxley2 / Darwin live departure boards.
